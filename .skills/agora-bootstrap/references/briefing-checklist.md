# Briefing Checklist

Before acting in a fresh Agora task session, make sure you can answer:

- Task id and title
- Controller agent id
- Your bound role
- Current stage id
- Current stage execution kind
- Allowed actions in this stage
- Task workspace path in the Agora AI Brain Pack
- Whether `04-context/project-brain-context.md` exists for this task
- Which `source_documents` are already in the current bootstrap context
- Which other agents are participating

If any of these are missing, ask the controller or inspect the task workspace before proceeding.
