# 09-PLANNING

This is the public planning surface for non-trivial contributions.

Use it when a change needs more than a quick PR description and benefits from explicit planning, findings capture, and progress logging.

## Structure

- [TASKS/README.md](./TASKS/README.md)
- [ROADMAP-ARCHIVE/README.md](./ROADMAP-ARCHIVE/README.md)

## Minimum Rule

For a non-trivial contribution, create:

- `Doc/09-PLANNING/TASKS/<YYYY-MM-DD-task-name>/task_plan.md`
- `Doc/09-PLANNING/TASKS/<YYYY-MM-DD-task-name>/findings.md`
- `Doc/09-PLANNING/TASKS/<YYYY-MM-DD-task-name>/progress.md`
