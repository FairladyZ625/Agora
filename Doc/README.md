# Agora Public Docs

This folder is the shareable documentation bundle that lives in the main repository.

It is intentionally smaller than the internal `docs/` repository. Only public-facing materials that are useful to external users should live here.

## Included now

- [whitepaper.md](./whitepaper.md)
- [quick-start.md](./quick-start.md)
- [architecture-overview.md](./architecture-overview.md)
- [discord-setup.md](./discord-setup.md)
- [openclaw-local-setup.md](./openclaw-local-setup.md)

## Scope rules

- Put tutorials, whitepapers, architecture overviews, and onboarding guides here.
- Keep planning logs, walkthroughs, internal reviews, and governance work in the separate `docs/` repo.
